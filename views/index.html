Request Header Parser

Usage:

  • /api/whoami - shows information about your IP address, language, and system software
  • /api/whoami-short - Deprecated; shows shortened information about your IP address, language, and system software
  • /api/whoami-full - Deprecated - shows full header information about your IP address, language, and system software

Example Output:

{ "ipaddress": "208.100.253.123", "language": "en-US,en;q=0.9,pt-BR;q=0.8,pt;q=0.7", "software": "Mozilla/5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/66.0.3359.181 Safari/537.36" }
